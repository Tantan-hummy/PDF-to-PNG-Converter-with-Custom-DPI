# PDF to PNG Converter with Custom DPI 📄✨

A lightweight, purely client-side web utility to split and convert PDF documents into high-resolution PNG images with precise, customizable DPI settings. 

[👉 Live Demo (Replace with your GitHub Pages URL)](https://Tantan-hummy.github.io/PDF-to-PNG-Converter-with-Custom-DPI/)

---

## 🌟 Features

* **Custom DPI Input:** Support standard resolutions (72, 150, 300, 600 DPI) as well as specific custom values (e.g., **350 DPI** for tailored printing workflows).
* **100% Private & Secure:** Powered by `pdf.js`. All file processing happens entirely inside your local browser. **No data is uploaded to any server.**
* **Batch ZIP Export:** Converts all pages automatically and bundles them into a single, clean ZIP file for effortless downloading.
* **Smart Multi-Language UI:** Automatically detects browser language and supports manual switching between **English**, **繁體中文**, **简体中文**, and **日本語**.

---

## 🚀 How to Host on GitHub Pages

1. **Create a GitHub repository** (e.g., `pdf-to-png-converter`).
2. Upload the `index.html` file to the root of the repository.
3. Go to **Settings** > **Pages** inside your repository.
4. Under **Build and deployment**, set the source to **Deploy from a branch**, choose `main` (or `master`) branch and `/root` folder, then click **Save**.
5. Your public link will be live within a few minutes!
